# Quick Reference

This page collects short computing references and templates used throughout the
notes.

## Command-line template

```bash
# Start a focused investigation
pwd
uname -a
whoami
```

## Performance checklist

- Establish a baseline before changing anything.
- Measure in production-like conditions.
- Attribute changes to one variable at a time.

## Notation

- `O(n)` indicates asymptotic time complexity.
- `latency` is time per request; `throughput` is requests per unit time.
- `contention` is resource competition between concurrent operations.
